# java-practice-programs

This repository contains a collection of beginner-friendly Java programs created while learning core Java concepts.

## Programs Included
- BMI Calculator  
- Factor Finder  
- Leap Year Checker  
- Number Checker  
- Quadratic Equation Solver  
- Random Statistics Generator  
- Sum of Natural Numbers  
- Unit Converter  

## Concepts Practiced
- Variables  
- Input/Output  
- Conditional Statements  
- Loops  
- Methods  
- Basic Math Logic  

Each file is a standalone Java program and can be compiled/run using:

javac FileName.java
java FileName
